## Menjalankan Project

### Menyalakan Docker

```bash
dcu 
```

### Menjalankan Migration

```bash
dca migrate
```

### Menjalankan Seeder

```bash
dca db:seed
```

### Fetch Data Cuaca

```bash
dca weather:fetch
```
